## Table of contents

- [Getting Started](#getting-started)
- [License](#license)

## Getting Started

To develop and run this documentation locally, proceed as following:

```
git clone https://github.com/deckgo/deckdeckgo
cd deckdeckgo/studio
npm install
npm run start
```
